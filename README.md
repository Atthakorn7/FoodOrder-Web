# 🍽️ Restaurant Ordering System (Demo)

โปรเจกต์เว็บสั่งอาหารสำหรับร้านอาหาร  
พัฒนาเพื่อใช้แทนการจดออเดอร์แบบเดิม  
ลูกค้าสามารถเลือกอาหารและสั่งผ่านหน้าเว็บ  
แอดมินสามารถจัดการเมนูและดูออเดอร์ที่เข้ามาได้แบบเรียลไทม์

---

## ✨ Features

### 👤 ลูกค้า (Customer)
- ดูรายการอาหารที่ร้านเปิดขาย
- เพิ่ม / ลด จำนวนอาหารในตะกร้า
- สั่งอาหารผ่านหน้าเว็บ
- ดูสถานะออเดอร์
  - กำลังเตรียมอาหาร
  - อาหารเสร็จแล้ว

### 👨‍🍳 แอดมิน (Admin)
- เพิ่ม / ลบ / เปิด–ปิด เมนูอาหาร
- ดูออเดอร์ทั้งหมดที่ลูกค้าสั่งเข้ามา
- เปลี่ยนสถานะออเดอร์ (กำลังทำ / เสร็จแล้ว)
- เมื่อแอดมินกด “เสร็จแล้ว”  
  หน้าลูกค้าจะแสดงว่าอาหารเสร็จทันที
